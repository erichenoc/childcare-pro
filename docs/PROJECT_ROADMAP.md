## Decisiones Técnicas

| Fecha | Decisión | Razón |
|-------|----------|-------|
| 2026-01-16 | OpenRouter + GPT-4o para agentes AI | Costo-efectivo, alta calidad |
| 2026-01-16 | Páginas estáticas por ciudad (SSG) | Mejor SEO que dynamic routing |
| 2026-01-16 | Schema markup LocalBusiness + SoftwareApplication | Mejora rich snippets en Google |
| 2026-01-16 | Route groups (marketing) para landing | Separación limpia de rutas |
| 2026-01-16 | Proactive chat después de 30s | Aumenta engagement sin ser intrusivo |
