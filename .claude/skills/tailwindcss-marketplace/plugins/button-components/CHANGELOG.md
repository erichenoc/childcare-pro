# Changelog

All notable changes to the Button Components plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-19

### Added
- Initial release of Button Components plugin
- 8 button variants (primary, secondary, success, danger, warning, ghost, outline, link)
- 5 size options (xs, sm, md, lg, xl)
- Interactive states (hover, focus, active, disabled, loading)
- Dark mode support for all variants
- Icon support with left/right positioning
- Button groups (horizontal and vertical)
- Ripple effect animations
- Full width button option
- Badge support for notifications
- WCAG-compliant accessibility features
- CSS variables for customization
- TypeScript definitions
- Comprehensive documentation and examples

### Features
- Focus-visible support for better keyboard navigation
- Smooth transitions and animations
- Gradient button support
- Custom color palette configuration
- Responsive button sizing
- Button group separators
- Loading spinner animation
- Disabled state styling

### Developer Experience
- Auto-completion support in VS Code
- Detailed configuration options
- Example components in React, Vue, and vanilla HTML
- Migration guide from other button libraries
- Performance optimizations with PurgeCSS
